SYUI pc端组件库
===========================
基于ElementUI二次封装的组件库

### 使用方法

> 因需要根据不同项目调整配置及样式，故不上传至npm

1、将components目录的sy-ui复制到项目的components目录下

2、在mian.js文件内引入组件库

```js
import '@/components/sy-ui'
```
